## Use Case 4 – Character Array Based Palindrome Check

## Description
This use case validates a palindrome by converting the string into a character array and comparing characters using the two-pointer technique.

## Functionality
- Converts the string into a character array
- Uses start and end pointers for comparison
- Checks characters from both ends moving inward
- Displays whether the string is a palindrome

## Output Example
Input: radar  
Result: true
