# Use Case 7 – Deque Based Optimized Palindrome Checker

## Description
This use case checks whether a string is a palindrome using a **Deque (Double Ended Queue)**.

A Deque allows insertion and removal from **both ends** of the structure.

Instead of reversing the string, characters are compared by removing:
- first element
- last element

This creates an efficient front-to-back comparison.
