# Use Case 5 – Stack Based Palindrome Checker

## Description
This use case validates whether a string is a palindrome using a **Stack data structure**.

A Stack follows the **LIFO (Last In First Out)** principle.  
By pushing characters into the stack and popping them, the string is reversed and compared with the original.

## Logic
1. Store the input string.
2. Push each character into a stack.
3. Pop characters from the stack.
4. Compare popped characters with the original string.
5. If all match → palindrome.
