# Use Case 2 – Hardcoded Palindrome Validation

## Description
This use case demonstrates basic palindrome validation using a predefined string.

## Functionality
- Stores a hardcoded input string
- Compares characters from both ends of the string
- Determines whether the string is a palindrome
- Displays the result on the console

## Class Implemented
- `UseCase2PalindromeCheckerApp`
