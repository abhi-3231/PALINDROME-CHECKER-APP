# Use Case 13 – Performance Comparison

## Description

This use case demonstrates how to measure the **execution time** of a palindrome validation algorithm.

The goal is to introduce basic **benchmarking concepts** in software development.

---

## Steps Performed

1. Capture the **start time** using `System.nanoTime()`
2. Execute the palindrome checking algorithm
3. Capture the **end time**
4. Calculate total execution time
