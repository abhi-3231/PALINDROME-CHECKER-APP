# Use Case 6 – Queue + Stack Palindrome Checker

## Description
This use case checks whether a string is a palindrome using two data structures:

- Queue (FIFO – First In First Out)
- Stack (LIFO – Last In First Out)

Characters from the input string are inserted into both structures.

Then characters are compared by removing:
- from the **front of the queue**
- from the **top of the stack**

If all characters match, the string is a palindrome.
